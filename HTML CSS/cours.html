Médias quaries

Les médias queries font partis des nouveautés de CSS3. Il ne s'agit pas de nouvelle propriétés mais de régles que l'on peut appliquer dans certaines conditions. Concrétement nous allons pouvoir dire si la résolution de l'écran de l'utilisateur est inférieur à tant, applique ces propriétés css. Cela nous permet de changer l'apparence du site en fontion de la tailles des écrans. Cest ce qu'on appelle la responsive design.

Syntaxte

Une requête médias se compose d'un type de médiz optionnel et d'un ou de plusieurs expressions de caractéristiques de média. Plusieurs requêtes peuvent être combinées entre elles grâce à des opérateurs logiques. Les requêtes médias ne sont pas sensibles à la casse.
Une requête média vaut true si le type de média correspond à l'appareil utilisé sinon cela vaut false.

Type de médias

Un type de média définie une catégorie général d'appareil. Le type de média est optionnel dans une requête média, sauf si celle-ci utilise les opérateurs logique not ou only. Par défaut, le type de média utilisé est le all.

  • all
    • Correspond à toutes les types de médias
  • print
    • Correspond aux médias paginé sur un support écrit
  • screen
    • Correspond aux appareils d'un écran
  • speach
    • Correspond aux appareils d'un systhèse volcale

Les caractéristiques médias

Elles décrivent certaines spécificitées de l'agent de l'appareil d'affichage ou de l'environnement. Les expressions des caractéristiques média testent leur présence ou leur valeur
Exemple:

  • Color
    • gestion de couleur de l'écran
  • with
    • gestion de largeur des écrans
  • eight
    • gestion de la hauteur d'affichage
  • ....

On peut rajouter les préfixe min et max devant la plupart de ces caractéristiques. Ainsi min-width voudra dire largeur minimale.

Opérateur logique

Les opérateurs logique not, only et and peuvent être utilisés afin de construire une requête média complexe. Il est aussi possible de combiner plusieurs requêtes média en les séparents par une virgule

  • and
    • il permet de combiner plusieurs requêtes médias
  • not
    • il est utilisé pour obtenir le résultat inverse de la requête média
  • only
    • il est utilisé afin d'appliquer un style unique à une requête
  • ,
    • il permet de combi,er plusieurs requêtes en une seule
